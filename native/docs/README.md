# Native 文档索引

本目录只保留当前 `native` 后端工程仍有维护价值的文档。历史报告、前端专项说明、已废弃方案和与当前代码不一致的内容已删除。

## 保留原则

- `swagger/` 是运行时依赖的 Swagger 生成包，不要手动删除。
- 规范、架构、功能、API、部署类文档保留在主目录。
- 完成报告、修复记录、评估文档、测试报告和历史迁移记录不保留在仓库文档中。
- 主目录文档应反映当前代码状态。

## Swagger

`swagger/` 由 `make swagger` 生成，并被 `cmd/api/main.go` 通过空导入加载：

```go
_ "github.com/gcc798/quick.admin/docs/swagger"
```

如果删除 `swagger/`，`go test ./...` 和构建会失败。需要更新接口文档时，在 `native` 目录执行：

```bash
make swagger
```

当前主目录保留 34 篇 Markdown 文档。
